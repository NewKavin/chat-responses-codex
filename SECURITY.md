# Security

If you find a security issue in `chat-responses-codex`, please report it privately instead of opening a public issue.

## Preferred Reporting Path

- Use GitHub Security Advisories if they are available for this repository.
- If that is not possible, open a private message to the maintainer or a private issue with the `security` prefix.

## What To Include

- A short description of the issue
- The affected endpoint or feature
- The expected impact
- Reproduction steps if you have them

Please do not include exploit details in a public issue.
